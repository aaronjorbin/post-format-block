Adds a Post Format Block as a variation of the core/post-terms block. 

== Description ==

Remember Post Formats? If you want to display a post format in a block based theme, you need a plugin and this one does that and only that.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/post-format-block` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress


== Frequently Asked Questions ==

= Can this display a post's tags? =

No, it can only display a post's format.

= What if the format is "Standard"? =

According to the WordPress Database, it thus doesn't have a format.
